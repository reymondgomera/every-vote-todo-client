# EveryVote Todo Client

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- React Query
- React Router
- Axios
- Zod
- Shadcn UI
- Jest & React Testing Library

## How to Run

```bash
npm install
npm run dev
```

## How to Test

```bash
npm run test
```

## Env

```bash
VITE_API_URL="http://localhost:8000/api/v1" # Back-end API URL
```
